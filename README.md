# 🎓 ISC Class 12 Marks Calculator (ICSE)

A simple and interactive web-based marks calculator designed for ISC (ICSE) Class 12 students to calculate total marks and percentage easily.

This project helps students quickly evaluate their academic performance by entering subject marks and getting instant results.

---

## 🚀 Features

- 📊 Calculate total marks
- 📈 Automatic percentage calculation
- ⚡ Instant result display
- 🧮 Simple and user-friendly interface
- 📱 Responsive design (works on mobile & desktop)
- 🎯 Designed specifically for ISC Class 12 pattern

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript

## 📌 How It Works

1. Enter marks for all required subjects.
2. Click on the **Calculate** button.
3. The system automatically:
   - Calculates total marks
   - Calculates percentage
   - Displays the final result
